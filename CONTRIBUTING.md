# Contributing to REPSHIELD

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Commit and push your changes
4. Open a Pull Request

## Areas We Need Help

- Voice safe word feature
- PIN/biometric vault lock
- Auto SMS without user interaction
- Sign language support for deaf users
- Firebase cloud backup
- Multi-language support

## Important

This app deals with personal safety.
Please test all features thoroughly
before submitting PRs.
